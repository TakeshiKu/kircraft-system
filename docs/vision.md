# Vision

## Цель системы

Создание цифровой системы управления заказами мастерской Kircraft.

## Пользователи

- Клиенты Telegram-бота
- Мастер

## Основные функции

- просмотр каталога изделий
- выбор параметров изделия
- оформление заказа
- подтверждение заказа мастером
- уточнение заказа
- оплата
- отправка заказа
