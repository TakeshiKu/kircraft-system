## UC-A6 Просмотр AI-summary переписки заказа

| Раздел | Содержание |
|------|-----------|
| Код | UC-A6 |
| Название | Просмотр AI-summary переписки заказа |
| Описание | Мастер открывает карточку заказа с длинной перепиской и получает краткое AI-резюме переписки для быстрого восстановления контекста. Фактическая переписка остаётся доступной в неизменном виде |
| Акторы | Мастер, Система, AI Assistant Module |
| Предусловия | Мастер аутентифицирован в бэкофисе. Заказ имеет переписку с клиентом. AI Assistant Module доступен |
| Основной сценарий | 1. Мастер открывает карточку заказа. |
|  | 2. Система отображает данные заказа и блок «Переписка». |
|  | 3. Мастер выбирает действие «Сформировать AI-summary переписки». |
|  | 4. Система передаёт переписку в AI Assistant Module в режиме чтения. |
|  | 5. AI Assistant Module формирует промпт и вызывает LLM Provider. |
|  | 6. AI Assistant Module логирует вызов: промпт, модель, версия, timestamp, результат, статус. |
|  | 7. Система отображает summary в карточке заказа с маркировкой «Сформировано AI». |
|  | 8. Мастер использует summary для восстановления контекста по заказу. |
|  | 9. Полная переписка остаётся доступной в карточке заказа без изменений. |
| Альтернативные сценарии | **3a. Переписка короче порогового объёма** |
|  | 3a1. Система не предлагает действие «Сформировать AI-summary». |
|  | 3a2. Мастер просматривает переписку напрямую. |
|  |  |
|  | **8a. Мастер обновляет summary после новых сообщений** |
|  | 8a1. Мастер выбирает действие «Обновить summary». |
|  | 8a2. Сценарий возвращается к шагу 4 с актуальной перепиской. |
| Исключения | LLM Provider недоступен или вернул ошибку. |
|  | В этом случае система отображает мастеру невозможность AI-операции и оставляет полную переписку без summary. Работа с карточкой заказа не блокируется. |
| Постусловия | В случае успеха — summary отображается в карточке заказа с маркировкой источника. Фактическая переписка не изменена. Вызов LLM зафиксирован в логе AI-вызовов. Данные заказа AI не изменялись |

Связанные BR: BR-AI-02, BR-AI-03, BR-AI-04, BR-AI-06.
